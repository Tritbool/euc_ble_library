ListenerConnectionCallback

class ListenerConnectionCallback(listener: BleBackendListener?, bleManager: BLEManager) : ConnectionCallback(source)

Default adapter that forwards ConnectionCallback events to a BleBackendListener.

Threading contract:

  • Forwarded events keep the original threading behavior of BLEManager.

  • Listener methods are therefore not guaranteed to run on the Android main thread.

Consumers of BleBackendListener must explicitly switch to the main thread before performing UI work.
