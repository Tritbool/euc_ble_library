class EucBleClient(context: Context, logger: Logger = AndroidLogger(), coroutineScope: CoroutineScope = CoroutineScope(Dispatchers.IO + SupervisorJob()))(source)

Main public entry point for EUC BLE discovery, connection and telemetry.

Protocol registration is fully managed internally; client applications should not select or register brand-specific protocols.
